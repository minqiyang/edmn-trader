# Handoff Archive

`docs/current_handoff.md` should stay compact and current. Do not let it become
a full history.

After each major stage:

1. Copy the previous `docs/current_handoff.md` into this directory.
2. Name the archive file `YYYY-MM-DD_stage-X.md`, replacing `X` with the stage
   label, such as `1-5` for Stage 1.5.
3. Update `docs/current_handoff.md` with only the latest state, commands,
   known issues, safety boundaries, next recommended stage, and exact next
   prompt suggestion.

Archive files are for historical continuity. The active handoff remains
`docs/current_handoff.md`.
